AntiCopias

El objetivo de este programa es disuadir a los alumnos de copiar. Puede ser más efectivo anunciar que se usa (y que funciona, y el tipo de manipulaciones que resiste) que el aplicar castigos ejemplares. Ningún sistema anticopias puede detectar todas las copias; pero es posible hacer que, para la gran mayoría, no resulte tentador copiar.

Aunque se puede ampliar facilmente, por defecto se usa el procedimiento de comparación "NCD", descrito en

R. Cilibrasi, P.M.B. Vitanyi, Clustering by compression, IEEE Trans. on Information Theory, 51:4(2005), 1523- 1545. http://xxx.lanl.gov/abs/cs.CV/031204 (2003).
